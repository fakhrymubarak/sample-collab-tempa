# sample-collab-tempa

## HTML & CSS Intro Project

A simple, modern, and visually appealing project explaining the basics of HTML and CSS. Features dark-mode aesthetics, glassmorphism, and smooth animations.

### How to Run

There are two simple ways to view this project:

#### 1. Direct Browser Viewing (Easiest)
Navigate to the `html-css-intro` directory and simply open the `index.html` file in your preferred web browser.
- **Mac:** Double-click the file or right-click `index.html` -> Open With -> Safari/Chrome/Firefox.
- **Windows:** Double-click the file or right-click `index.html` -> Open with -> Chrome/Edge/Firefox.
- Alternatively, drag and drop the `index.html` file right into an open browser tab.

#### 2. Using a Local Server (Recommended for development)
If you have Python installed, you can serve the directory locally. Open your terminal, navigate to the project directory, and run:

```bash
cd html-css-intro
python3 -m http.server 8000
```
Then, open your browser and navigate to `http://localhost:8000`.

Alternatively, if you have Node.js installed, you can use `npx serve`:
```bash
cd html-css-intro
npx serve .
```
